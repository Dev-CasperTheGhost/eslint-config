# eslint-config-react

## Install

```bash
npm install @casper124578/eslint-config @casper124578/eslint-config-react
```

## Usage

Use `extends` in your ESLint config

```json
{
  "extends": ["@casper124578/eslint-config", "@casper124578/eslint-config-react"]
}
```
